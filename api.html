TV Show Search

About This API

API Selected: TVMaze Show Search API

Why I Selected It: I chose it because it is simple to use, free, and lets users search for TV shows by name.

Where I Found It: TVMaze API documentation page.

Parameters Used: I used the q parameter in the query string. This is the search term entered by the user, such as ?q=friends.

Helpful Use Case: This API would be useful for building a movie/TV discovery website where users can quickly search for shows and view information like genres, ratings, and summaries.
